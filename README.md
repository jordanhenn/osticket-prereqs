<h1>osTicket - Prerequisites and Installation</h1>
This tutorial outlines the prerequisites and installation of the open-source help desk ticketing system osTicket.<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>List of Prerequisites</h2>

- Microsoft Azure
- Virtual Machine
- Web Platform Installer
- PHP Version 7.3.8
- PHP Manager 1.5.0 for IIS 10
- Microsoft Visual C++ 2009 Redistributable Package
- osTicket v1.15.8

<h2>Installation Steps</h2>

<h3>Step 1: Connect to Virtual Machine</h3>

- In Azure, create a resource group named 'RG-osTicket'
- Create a Virtual Machine and include it in the RG-osTicket resource group
- Select Windows 10 as the Image for the Virtual Machine
- Set Size of VM to 4 vcpus, 16 GiB memory
- Create a username and password for the VM 
- Proceed to the next steps of creating a VM until you get to the 'Networking' menu
- Allow a new Virtual Network to be created for the Virtual Machine
- Review & Create the VM
- Remote Desktop Connect into the VM using it's public IP address and the username and password you created earlier
